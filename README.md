# Simple gulp project

## Installation

`$ npm install `

## Command

#### Default gulp task
`$ gulp`
#### Clean generated, build sass ans js
`$ gulp build`
#### Build sass ans js with file watcher
`$ gulp watch`
#### Build sass to css
`$ gulp sass `
#### Build js 
`$ gulp js `
#### Build js loose files
`$ gulp jsloose
#### Clean generated folder
`$ gulp clean`
